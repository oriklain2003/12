**Create `PolygonMapWidget.tsx`:**

Props: `{ initialPolygon?: number[][], onConfirm: (polygon: number[][]) => void, onClose: () => void }`

Component renders a `position: fixed` overlay (z-index: 9999) covering the full viewport with a semi-transparent dark backdrop. Inside:
- A Leaflet `MapContainer` (center [32, 35], zoom 6, style height 70vh width 80vw centered) with CartoDB dark tiles (`https://{s}.basemaps.cartocdn.com/dark_all/{z}/{x}/{y}{r}.png`)
- A `ClickCapture` child component using `useMapEvents({ click(e) { addPoint(e.latlng.lat, e.latlng.lng) } })` to capture map clicks
- `Polyline` rendering the current points (close the polygon visually by appending first point to positions when 3+ points)
- `CircleMarker` for each vertex (radius 6, accent color fill)
- A control bar at the bottom with: "Clear" button (resets points), "Cancel" button (calls onClose), "Confirm (N pts)" button (calls onConfirm(points) then onClose, disabled if < 3 points)
- Apply `nodrag nowheel` classes and `onClick={e => e.stopPropagation()}` on the overlay div to prevent React Flow canvas interference

Local state: `const [points, setPoints] = useState<number[][]>(initialPolygon ?? [])`
