  <action>
Create `backend/app/cubes/get_flight_course.py` implementing `GetFlightCourseCube(BaseCube)`.

**Class attributes:**
- `cube_id = "get_flight_course"`
- `name = "Get Flight Course"`
- `description = "Get flight track points or lines from normal_tracks"`
- `category = CubeCategory.DATA_SOURCE`

**Inputs:**
- `flight_ids` (list_of_strings, required) — flight IDs to query
- `output_mode` (string, required, default="points") — `"points"` or `"lines"`

**Outputs:**
- `tracks` (json_object) — array of track data rows
- `flight_ids` (list_of_strings) — unique flight IDs in results

**execute() logic:**

1. Extract `flight_ids` and `output_mode` from inputs. Guard empty flight_ids early (return empty results to avoid PostgreSQL ANY() error with empty array — same pattern as GetAnomalies).

2. Query `research.normal_tracks`:
   ```sql
   SELECT flight_id, timestamp, lat, lon, alt, gspeed, vspeed, track, squawk, callsign, source
   FROM research.normal_tracks
   WHERE flight_id = ANY(:flight_ids)
   ORDER BY flight_id, timestamp
   ```

3. **Points mode** (`output_mode == "points"` or default):
   - For each row, add a `geometry` field with GeoJSON Point: `{"type": "Point", "coordinates": [lon, lat]}` (GeoJSON uses [lon, lat] order). Skip rows where lat/lon is None.
   - Return `{"tracks": rows, "flight_ids": unique_flight_ids}`

4. **Lines mode** (`output_mode == "lines"`):
   - Group rows by `flight_id` using `collections.defaultdict`
   - Sort each group by timestamp
   - For each flight, build a GeoJSON LineString from ordered `[lon, lat]` coords (skip flights with fewer than 2 valid coordinate points)
   - Each output row: `flight_id`, `callsign` (from first point), `geometry` (LineString), `start_time` (first timestamp), `end_time` (last timestamp), `min_alt`, `max_alt` (min/max of non-None alt values)
   - Return `{"tracks": lines, "flight_ids": [line["flight_id"] for line in lines]}`

Use `engine.connect()` directly for DB access (established pattern). Import `text` from sqlalchemy. Use `collections.defaultdict` for grouping.
