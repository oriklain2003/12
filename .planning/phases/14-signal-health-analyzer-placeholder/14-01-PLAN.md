4. `classify_event(scored_event)` -> keep as sync `classify_event(scored_event) -> str`
   - Pure computation mapping scores to categories
   - Categories: gps_jamming, probable_jamming, gps_spoofing, transponder_off, coverage_hole, ambiguous

5. `build_coverage_baseline(lookback_days)` -> `async def build_coverage_baseline_async(lookback_days=7) -> dict`
   - 0.5-degree grid with median RSSI, reports/hour, temporal coverage
   - NOTE: Default to 7 days (not 30) for interactive cube use per RESEARCH.md recommendation
   - Implement module-level TTL cache (1 hour) per Pattern 2 in RESEARCH.md:
     ```python
     _baseline_cache = None
     _baseline_ts = 0.0
     _BASELINE_TTL = 3600.0

     async def get_coverage_baseline(lookback_days=7):
         global _baseline_cache, _baseline_ts
         now = time.monotonic()
         if _baseline_cache is None or (now - _baseline_ts) > _BASELINE_TTL:
             _baseline_cache = await build_coverage_baseline_async(lookback_days)
             _baseline_ts = now
         return _baseline_cache
     ```

6. Any helper functions used by the above (haversine_km, grid coordinate rounding, etc.) — port as-is (pure Python, no changes needed).
