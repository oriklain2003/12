<objective>
Add SSE streaming execution to the workflow engine: a new `stream_graph` async generator that yields per-cube status events, and a `GET /api/workflows/{id}/run/stream` endpoint that wraps it with `EventSourceResponse`.

Purpose: Enables real-time per-cube progress updates during workflow execution. The frontend (Phase 5) will consume these SSE events to show live status indicators on each cube node.

Output: CubeStatusEvent schema, stream_graph generator, SSE route handler, tests proving real-time streaming behavior.
</objective>
