**Inner MapController component** (MUST be inside MapContainer — Pitfall 4):
- Uses `useMap()` hook
- `useEffect` watching `selectedRowIndex`: when non-null, get lat/lon from `rows[selectedRowIndex]` using `geoInfo.latCol` and `geoInfo.lonCol`. If both are finite numbers, call `map.flyTo([lat, lon], 10, { animate: true, duration: 0.5 })`.

**Inner MapBoundsController component** (auto-fit bounds on data load per RESEARCH.md recommendation):
- Uses `useMap()` hook
- `useEffect` watching `rows` and `geoInfo`: compute bounds from all valid lat/lon pairs in rows. If any valid points exist, call `map.fitBounds(bounds, { padding: [30, 30] })`. This runs once when data loads to frame all markers.

**buildGeoJSON helper function** (not exported, module-internal):
- Takes `rows: unknown[]` and `geoInfo: GeoInfo`, returns `GeoJSON.FeatureCollection`
- For each row, extract lat/lon. Skip if not finite numbers.
- If `geoInfo.geomCol` exists and the row has a valid object at that key, use it as the geometry directly (supports polygon/shape data per user decision).
- Otherwise create a Point geometry with `[lon, lat]` coordinates (GeoJSON uses lon,lat order).
- Each Feature gets `properties: { rowIndex: i }` for click mapping.

**Main ResultsMap component:**
- `geojson` computed via `useMemo(() => buildGeoJSON(rows, geoInfo), [rows, geoInfo])`
- Render `MapContainer` with center `[20, 0]`, zoom `2`, className `"results-map"`, zoomControl true
- Do NOT put a `key` on MapContainer (Pitfall 3 — prevents flash on cube switch). ONLY key the GeoJSON layer.
- CartoDB dark tiles TileLayer: url `"https://{s}.basemaps.cartocdn.com/dark_all/{z}/{x}/{y}.png"`, subdomains `['a','b','c','d']`, maxZoom 19, attribution including CARTO
- GeoJSON component with `key={JSON.stringify(geojson)}` to force re-mount on data change (Pitfall 2). Configure:
  - `pointToLayer`: return `L.circleMarker(latlng, { radius: 6, fillColor: '#6366f1', color: '#818cf8', weight: 1.5, opacity: 1, fillOpacity: 0.7 })` — circle markers per user decision, NOT generic pin markers
  - `onEachFeature`: `layer.on('click', () => onMarkerClick(feature.properties.rowIndex))` — click marker calls onMarkerClick
  - `style` for non-Point geometries: `{ color: '#6366f1', fillColor: '#6366f1', weight: 1.5, fillOpacity: 0.3 }` — render shapes per user decision
- Render `MapController` and `MapBoundsController` as children of MapContainer

**1. Create `frontend/src/components/Results/ResultsDrawer.tsx`:**

Imports: ResultsTable, ResultsMap, detectGeoColumns, GeoInfo, useFlowStore.

The drawer is the orchestrator component. Selection state is LOCAL (useState), NOT in Zustand (per RESEARCH.md anti-pattern guidance — ephemeral view state).

State:
- `selectedRowIndex: number | null` via useState(null) — shared between table and map
- `splitRatio: number` via useState(0.55) — table gets 55% by default (Claude's discretion)
- Read from store: `selectedResultNodeId` and `results[selectedResultNodeId]`

Computed:
- `geoInfo = useMemo(() => results ? detectGeoColumns(results.rows) : null, [results])`
- `isOpen = selectedNodeId !== null && results !== null && results.rows.length > 0`

Reset selectedRowIndex when the selected cube changes: `useEffect(() => { setSelectedRowIndex(null); }, [selectedNodeId])`

**ResizeDivider** (inline component or function within the file):
- Takes `containerRef: React.RefObject<HTMLDivElement>` and `onSplitChange: (ratio: number) => void`
- `isDragging` via useRef(false)
- onPointerDown: set isDragging true, setPointerCapture
- onPointerMove: if dragging, compute ratio from `(e.clientX - rect.left) / rect.width`, clamp to [0.15, 0.85], call onSplitChange
- onPointerUp: set isDragging false, releasePointerCapture
- Render a `div.results-drawer__divider`

**Drawer JSX structure:**
```
<div className={`results-drawer ${isOpen ? 'results-drawer--open' : ''}`}>
  <div className="results-drawer__handle" onClick={() => setSelectedResultNodeId(null)}>
    {/* Grip line or close indicator — discretion */}
    <div className="results-drawer__grip" />
  </div>
  <div className="results-drawer__header">
    <span className="results-drawer__title">{cubeName} Results</span>
    <button className="results-drawer__close" onClick={() => setSelectedResultNodeId(null)}>Close</button>
  </div>
  <div className="results-drawer__content" ref={contentRef}>
    <div className="results-drawer__table-pane" style={{ flex: geoInfo ? `0 0 ${splitRatio * 100}%` : '1' }}>
      <ResultsTable rows={results.rows} truncated={results.truncated} selectedRowIndex={selectedRowIndex} onRowSelect={setSelectedRowIndex} />
    </div>
    {geoInfo && (
      <>
        <ResizeDivider containerRef={contentRef} onSplitChange={setSplitRatio} />
        <div className="results-drawer__map-pane" style={{ flex: 1 }}>
          <ResultsMap rows={results.rows} geoInfo={geoInfo} selectedRowIndex={selectedRowIndex} onMarkerClick={setSelectedRowIndex} />
        </div>
      </>
    )}
  </div>
</div>
```

For the cube name in the header, look up the node from the store: `const nodeName = useFlowStore(s => { const node = s.nodes.find(n => n.id === selectedNodeId); return node?.data.cubeDef.name ?? 'Results'; });`
