**3. Restructure `execute()` — replace the per-hex semaphore + _analyze_hex pattern:**

The new execute() flow:

```
Step 1: Extract hex_list (unchanged)
Step 2: Compute time range in Python:
    end_ts = datetime.now(timezone.utc)
    start_ts = end_ts - timedelta(hours=lookback_hours)
Step 3: Get coverage baseline (unchanged: await get_coverage_baseline())
Step 4: Batch queries — 3 total, run concurrently:
    integrity_by_hex, shutdown_by_hex, positions_by_hex = await asyncio.gather(
        detect_integrity_events_batch_async(hex_list, start_ts, end_ts),
        detect_shutdowns_batch_async(hex_list, start_ts, end_ts),
        fetch_positions_batch_async(hex_list, start_ts, end_ts),
    )
Step 5: Per-hex processing loop (in-memory, no DB):
    For each hex in hex_list:
      - Score + classify integrity events from integrity_by_hex.get(hex, [])
      - Score + classify shutdown events from shutdown_by_hex.get(hex, [])
      - Run Kalman on positions_by_hex.get(hex, []) via classify_flight_async(hex, start_ts, end_ts, positions=positions)
      - If Kalman classification is not normal/None, create kalman event via kalman_event_from_result
      - Apply target_phase filter
      - Collect events per hex
Step 6: Apply classify_mode filtering (unchanged logic)
Step 7: Build outputs (unchanged logic)
```

**Important details for Step 5:**
- Normalize each hex: `hx = str(raw_hex).strip().lower()`
- Wrap individual hex processing in try/except, log warning and skip on error (same error handling as before)
- For Kalman: if `positions_by_hex.get(hx)` is empty or None, skip Kalman for that hex
- Pass `positions=hex_positions` to `classify_flight_async` so it skips the per-hex DB fetch
- `classify_flight_async` is still async (uses run_in_executor internally) so `await` it

**4. Remove `_analyze_hex()` method entirely.**

**5. Remove `asyncio.Semaphore` — no longer needed since batch queries use constant connections.**

**6. Keep `_filter_events_by_phase()` and `_empty_result()` unchanged.**

**7. Keep all cube metadata (cube_id, name, description, category, inputs, outputs) unchanged.**

Code quality: The execute() method should be clean and linear — no nested async closures, no semaphores. The per-hex loop is just Python dict lookups and sync scoring/classification.
